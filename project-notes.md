# portal-numeral-topographic-field

## Competition
Google I/O 2026 — Code the Countdown

## Core idea
A giant numeral acts as a portal into a living monochrome topographic field.

## Creative direction
- Minimal dark background
- Strong numeral legibility
- Cartographic contour lines
- Distinct structural logic for different digits
- Elegant, broadcast-friendly motion
- No generic RGB glitch, cyberpunk effects, or noisy particle clouds

## Technical approach
- Single HTML file
- Full-screen WebGL canvas
- Text mask overlay for the numeral
- Digit selector from 1 to 10
- Mouse interaction for pressure and parallax
- Fallback mode if WebGL is unavailable

## Iteration summary
Version 1:
- Base WebGL portal effect
- Distinct mathematical structures per digit
- Simple contour-line logic

Version 2:
- Improved fallback handling
- Sharper cartographic major/minor contour lines
- Stronger differences for key digits (1, 3, 7, 10)
- Reduced noise contribution for a cleaner topographic identity

## Final short description
A huge central numeral reveals a living topographic field inside its shape, with distinct visual behavior for digits 1–10 and subtle interaction through mouse motion.
